# State Cell SOP

## Objective

Define how state patient safety intelligence cells receive, review, investigate, and escalate surveillance information.

## Core tasks

1. review facility submissions daily or weekly depending on program maturity,
2. confirm triage and severity classification,
3. assign investigators for sentinel and high-severity cases,
4. monitor district and facility closure performance,
5. escalate unusual clusters to the national node,
6. coordinate state learning collaboratives.
