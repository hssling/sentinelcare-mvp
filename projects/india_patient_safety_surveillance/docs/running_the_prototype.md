# Running the Prototype

## Backend

1. install the project root package:
   - `pip install .`
2. run the API:
   - `india-surveillance-api`
3. run the demo CLI:
   - `india-surveillance-demo`

## Frontend

1. `cd frontend/india-surveillance`
2. `npm install`
3. `npm run dev`

Optional frontend API environment variable:

- `VITE_INDIA_SURVEILLANCE_API_BASE=http://127.0.0.1:8010`
