# IJMR Companion Review Package

This folder contains an IJMR-compatible companion manuscript package focused on patient safety reporting, learning systems, and implementation lessons relevant to India.

## Proposed article type

- `Special Article`
- review-oriented companion manuscript

## Companion manuscript focus

This paper complements the federated surveillance framework paper by synthesizing:

1. the strengths and limitations of incident reporting systems
2. the conditions under which reporting becomes organizational learning
3. the implementation implications for India
