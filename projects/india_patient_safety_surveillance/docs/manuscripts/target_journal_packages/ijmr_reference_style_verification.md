# IJMR reference-style verification note

## Verification basis

Reference and submission formatting were checked against:

1. IJMR author guidance page
2. IJMR author instructions PDF
3. recent IJMR article pages used as live style examples

## Normalization decisions applied in this repository

1. numbered citation system retained throughout IJMR-targeted manuscripts
2. in-text citations normalized toward IJMR-style numbered superscript usage in the markdown source
3. reference lists kept in Vancouver-style order of appearance
4. journal titles retained in abbreviated biomedical style where applicable
5. author lists kept in standard indexed format with `et al.` for longer multi-author citations

## Final manual checks still recommended at submission stage

1. compare punctuation and author-separator details with the latest published IJMR examples in Word
2. verify whether the final uploaded manuscript should keep superscripts or whether the production office will normalize them automatically
3. verify web-source formatting for government and WHO documents during the final editorial pass
