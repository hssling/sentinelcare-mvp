# Portal upload order for IJMR round 1

Upload in this order unless the journal portal explicitly requests a different sequence:

1. title page
2. blinded manuscript
3. cover letter
4. summary table
5. figure package

## Metadata to confirm during upload

1. article category: `Special Article`
2. running title
3. corresponding author details
4. keywords
5. no ethics approval required because no primary human-participant study data were analyzed

## Blinding rule

Do not upload the non-blinded manuscript in place of the blinded manuscript.
