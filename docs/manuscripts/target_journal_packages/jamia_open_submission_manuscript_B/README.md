# JAMIA Open Submission Package for SentinelCare Manuscript B

This package converts the multidomain taxonomy and process-deviation manuscript into a `JAMIA Open` `Research and Applications` submission candidate.

## Focus

Manuscript B defines the cross-domain semantic substrate for SentinelCare: domain taxonomy, process-deviation classes, common alert semantics, and cross-domain metrics.

## Rationale for journal fit

`JAMIA Open` is appropriate because the manuscript reports the formulation of a new informatics model with operational and implementation implications for clinical decision support and patient safety.
