# JAMIA Open Submission Checklist

## Article type

- `Research and Applications`

## Current instruction alignment

- Structured abstract uses required headings:
  - `Objectives`
  - `Materials and Methods`
  - `Results`
  - `Discussion`
  - `Conclusion`
- Main text includes required `Background and Significance` section.
- Main text also includes:
  - `Objectives`
  - `Materials and Methods`
  - `Results`
  - `Discussion`
  - `Conclusion`
- Main text target is within the journal's 4000-word limit excluding references and acknowledgments.
- Tables prepared: `4` of maximum `4`.
- Figures prepared: `3` of maximum `6`.
- References: unlimited.

## Additional journal requirements reflected in package

- Separate title page prepared.
- Cover letter prepared.
- Data availability statement included.
- Code repository included.
- Patient/community-facing abstract prepared in advance.
- AI-use disclosure template included because journal policy requires disclosure if AI assistance was used.

## Author actions still required before submission

1. Replace all placeholder author, affiliation, funding, and conflict fields.
2. Confirm whether prior presentation or preprint posting applies.
3. Verify every reference against the final target style in the submission system.
4. Export final figures in submission-ready graphic format.
5. Add final human-authored AI disclosure wording if applicable.
