## Required checks before submission

1. Title <=15 words.
2. Abstract <=150 words, no subheadings.
3. Main sections include:
   - Introduction
   - Results
   - Discussion
   - Methods
4. Mandatory statements included:
   - Data availability
   - Code availability
   - Competing interests
   - Author contributions
5. Reference count <=60 recommended.
6. Figure/table files prepared separately (if included).
7. Cover letter finalized with corresponding author details.
8. Reporting summary/editorial checklist prepared per Nature submission portal prompts.

## Project-specific completion items

1. Replace all placeholders in title page and author sections.
2. Finalize ethical/IRB language for study stage and data use.
3. Expand references to full journal style formatting.
4. Add architecture figure and validation flow figure.
5. Lock manuscript word counts and consistency review.
