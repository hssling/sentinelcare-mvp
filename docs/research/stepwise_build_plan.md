# Stepwise Build Plan: Theory to Product

## Phase 0: Concept and evidence baseline

1. Define harm model and process taxonomy.
2. Map MVP scope to external guidelines.
3. Draft protocol manuscript and architecture manuscript.

## Phase 1: Technical prototype (current)

1. Event ingestion and safety engines.
2. Multi-agent orchestration tasks.
3. Routing, queueing, review actions.
4. Basic governance and validation APIs.

## Phase 2: Clinical pilot readiness

1. Site data mapping and interoperability adapters.
2. Role-owner matrix and escalation SLA policy finalization.
3. Silent-mode reporting dashboards.
4. Pilot IRB/ethics package and training SOP.

## Phase 3: Controlled live pilot

1. Restricted unit deployment.
2. Weekly governance board review.
3. Alert burden and trust calibration.
4. Outcome and process KPI capture.

## Phase 4: Manuscript and scale package

1. Pilot outcome manuscript.
2. Validation appendix and model cards.
3. Multi-site scaling design and grant/funding package.
